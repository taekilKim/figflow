# FigFlow 빠른 시작 가이드

이 문서는 FigFlow 프로토타입을 5분 안에 실행하고 체험하는 방법을 안내합니다.

## 1단계: 프로젝트 실행하기 (2분)

```bash
# 터미널에서 프로젝트 디렉토리로 이동
cd figflow

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

서버가 시작되면 자동으로 브라우저가 열리거나, 다음 주소로 접속:
```
http://localhost:5173
```

## 2단계: 데모 체험하기 (3분)

### 화면 구성

프로토타입은 3개 패널로 구성되어 있습니다:

```
┌──────────────┬────────────────────┬──────────────┐
│  프레임 목록  │   React Flow       │  속성 패널    │
│  (좌측)      │   캔버스 (중앙)     │  (우측)      │
│              │                    │              │
│  - 검색      │  - 노드 3개         │  - 노드 정보  │
│  - 목록      │  - 연결선 2개       │  - 엣지 정보  │
│              │  - 줌/팬           │              │
└──────────────┴────────────────────┴──────────────┘
```

### 체험해볼 기능

#### ✅ 노드 조작
1. **드래그**: 노드를 클릭하고 드래그해서 위치 변경
2. **선택**: 노드 클릭 → 우측 패널에 속성 표시
3. **연결**: 노드 상단/하단의 작은 원(핸들)을 드래그해서 다른 노드와 연결

#### ✅ 연결선 조작
1. **선택**: 연결선 클릭 → 우측 패널에 라벨 입력란 표시
2. **라벨 편집**: 우측 패널에서 "라벨" 필드에 텍스트 입력
3. **삭제**: 연결선 선택 후 Delete 키

#### ✅ 캔버스 조작
1. **줌 in/out**: 마우스 휠 스크롤
2. **팬 (이동)**: 빈 공간을 클릭하고 드래그
3. **전체 보기**: 우측 하단 미니맵으로 전체 뷰 확인

#### ✅ 상태 관리
1. 노드 선택
2. 우측 패널에서 "상태" 드롭다운 변경
   - **초안** (회색)
   - **검토 중** (주황색)
   - **승인됨** (녹색)
   - **사용 중지** (빨간색)

#### ✅ 메모 추가
1. 노드 또는 연결선 선택
2. 우측 패널에서 "메모" 텍스트 영역에 입력

#### ✅ 저장하기
상단 툴바의 **"저장"** 버튼 클릭
- 프로젝트가 브라우저 localStorage에 저장됨
- 페이지를 새로고침해도 데이터 유지됨

## 3단계: Figma 연동하기 (선택사항)

실제 Figma 프레임의 썸네일을 가져오려면:

### Figma Personal Access Token 발급

1. https://www.figma.com/developers/api#access-tokens 접속
2. "Generate new token" 클릭
3. 토큰 이름 입력 (예: "FigFlow Test")
4. 생성된 토큰 복사 ⚠️ **한 번만 표시되므로 반드시 저장!**

### FigFlow에서 Sync 실행

1. 상단 툴바의 **"Sync"** 버튼 클릭
2. 팝업창에 복사한 토큰 붙여넣기
3. 확인 클릭

샘플 노드가 있는 Figma 파일에 접근 권한이 있다면 썸네일이 자동으로 업데이트됩니다.

## 문제 해결

### 포트가 이미 사용 중일 때
```bash
# Vite가 자동으로 다른 포트(5174, 5175 등)를 사용합니다
# 터미널에 표시된 URL로 접속하세요
```

### 노드가 보이지 않을 때
- 브라우저 콘솔 (F12) 확인
- localStorage 초기화: 브라우저 개발자 도구 → Application → Local Storage → 삭제 후 새로고침

### Sync가 작동하지 않을 때
- Figma Access Token이 올바른지 확인
- 샘플 노드의 fileKey가 실제 접근 가능한 Figma 파일인지 확인
- 브라우저 콘솔에서 에러 메시지 확인

## 자신의 Figma 프레임 추가하기

현재 MVP에서는 코드를 직접 수정하여 프레임을 추가할 수 있습니다:

1. Figma에서 프레임 선택 → 우클릭 → "Copy link to selection"
2. URL 예시: `https://www.figma.com/file/ABC123/MyFile?node-id=123-456`
3. `src/components/FlowCanvas.tsx` 열기
4. `initialNodes` 배열에 새 노드 추가:

```typescript
{
  id: '4',  // 고유 ID
  type: 'frameNode',
  position: { x: 100, y: 400 },
  data: {
    figma: {
      fileKey: 'ABC123',           // URL에서 추출
      nodeId: '123:456',            // node-id를 : 로 변환
      nodeUrl: 'https://www.figma.com/file/ABC123/MyFile?node-id=123-456',
    },
    meta: {
      title: '내 프레임 이름',
      status: 'draft',
      notes: '설명 추가',
    },
  },
}
```

5. 서버 재시작 (Ctrl+C 후 `npm run dev`)
6. "Sync" 버튼 클릭하여 썸네일 가져오기
